# 精选艺术图集（Resonance Calendar 背景图）

为了保证“质感”和文字可读性，推荐使用**本地精选图片库**（低饱和、干净、留白多的艺术摄影/纹理）。

## 推荐素材规范

- 风格：温柔、治愈、低饱和；避免主体过多、文字区域太花
- 方向：竖图更佳（适配明信片 3:4）
- 分辨率：建议 ≥ 1080×1440（越高越好；下载 PNG 会更清晰）
- 体积：单张尽量 ≤ 800KB（避免移动端加载慢）

## 如何接入

1. 把图片放进本目录，例如：
   - `calendar-backgrounds/01.jpg`
   - `calendar-backgrounds/02.jpg`
2. 打开 `index.html`，找到 `CURATED_BACKGROUND_IMAGES` 数组，把文件名加入即可。

> 说明：纯静态网页无法自动读取文件夹内容，因此需要手动维护数组。
