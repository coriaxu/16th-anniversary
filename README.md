# 💕 16 周年恋爱纪念日

> 致 Minnie：我们的时光档案

一个精心设计的恋爱纪念日网页，记录我们从 2009 年 12 月 10 日开始的爱情故事。

## ✨ 预览

🔗 **在线访问**: [点击查看](https://coriaxu.github.io/16th-anniversary/)

## 🌟 功能特点

- 💗 **实时计时器** - 显示在一起的天数、小时、分钟、秒数
- 💓 **心跳计数** - 估算两人共同的心跳次数
- 🎵 **背景音乐** - 《遇见》钢琴版，点击进入后自动播放
- 🎉 **撒花特效** - 进入页面和双击时的庆祝动画
- 🌹 **倒计时** - 距离第 17 年玫瑰婚的天数
- 🎨 **主题切换** - 纪念日当天自动切换特别主题
- 📱 **响应式设计** - 完美适配手机、平板、电脑

## 🛠️ 技术栈

- HTML5
- CSS3 (TailwindCSS)
- Vanilla JavaScript
- Canvas Confetti 撒花特效
- Google Fonts 字体
