# 💕 16 周年恋爱纪念日

> 致 Minnie：我们的时光档案

一个精心设计的恋爱纪念日网页，记录我们从 2009 年 12 月 10 日开始的爱情故事。

## 🤖 我能做什么 / 不能做什么（用初中生能懂的方式说）

- ✅ 我能做的：
  - 改网页的文字、颜色、布局，让页面更好看；
  - 写简单好懂的 HTML / CSS / JS 代码，并加中文注释；
  - 帮你检查网页显示问题、解释代码是干嘛的；
  - 把修改过的文件提交到当前项目里，生成拉取请求说明。
- 🚫 我不能做的：
  - 访问互联网、登录你的网站后台或第三方账号；
  - 运行需要外部服务或硬件支持的东西（比如摄像头、麦克风实时采集）；
  - 做出违反安全或法律的操作。

一句话总结：我能在这个项目里写代码、改样式、解释问题，但不能联网或帮你操作外部账号。

### 冲突处理 & 部署确认（步骤清单）

1. **拉取主分支最新代码**：`git pull origin main`，确保本地与线上一致。
2. **解决冲突再提交**：若提示有冲突，手动比对并修改后执行 `git add ... && git commit`。
3. **合并到主分支**：将修复后的分支合并回主分支并推送；GitHub Pages 会在合并后自动重新部署。
4. **验证上线**：刷新浏览器或开无痕模式查看城市卡片，确认厦门文案显示“集美街头大快朵颐闽南小吃”即为成功。如果仍旧显示旧文案，大概率是缓存；可再等待几分钟或清缓存重试。

## 🌟 功能特点

- 💗 **实时计时器** - 显示在一起的天数、小时、分钟、秒数
- 💓 **心跳计数** - 估算两人共同的心跳次数
- 🎵 **背景音乐** - 《遇见》钢琴版，点击进入后自动播放
- 🎉 **撒花特效** - 进入页面和双击时的庆祝动画
- 🌹 **倒计时** - 距离第 17 年玫瑰婚的天数
- 🎨 **主题切换** - 纪念日当天自动切换特别主题
- 📱 **响应式设计** - 完美适配手机、平板、电脑

## 🛠️ 技术栈

- HTML5
- CSS3 (TailwindCSS)
- Vanilla JavaScript
- Canvas Confetti 撒花特效
- Google Fonts 字体
